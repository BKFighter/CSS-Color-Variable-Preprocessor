CSS Color Variable Preprocessor - CCV









What is CCV?

CCV (CSS Color Variable Preprocessor) is a preprocessor that allows you to make a regular css template but be able to change the colors to fit different projects. Instead of manually changing every color, you can use CCV to pick a different main and secondary color while loading the page.

Want a demo?

How can I use this?

Easy! Four steps are required.

1. When creating your css document, use the file extension .ppcss, to ensure that it is not treatd like a regular css file, but so that the user is aware of its purpose.

2. Anywhere in the code you want the color to be changed, either type "+main+" or "+secondary+".

3. Place this script tag <script src="https://raw.githubusercontent.com/BKFighter/CSS-Color-Variable-Preprocessor/gh-pages/CCV-Preprocessor.js"> in the head of the page.

4. Also in the head, place this script. process(url,main,secondary); Replace url with the link to your .ppcss file. (Which should either be stored in the same repository as the website, or accessed using a service like raw.githubusercontent, etc.) Replace main with the main color of your theme, and replace secondary with the second color of your theme. Make sure surround each of these in quotation marks (").

Now you can pick different theme colors while reusing the same template. You could even pick different theme colors dynamically as the page loads.
