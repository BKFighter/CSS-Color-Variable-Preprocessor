CSS Color Variable Preprocessor - CCV









What is CCV?

CCV (CSS Color Variable Preprocessor) is a preprocessor that allows you to make a regular css template but be able to change the colors to fit different projects. Instead of manually changing every color, you can use CCV to pick a different main and secondary color while loading the page.

How can I use this?

Easy! Three steps are required.

1.When creating your css document, use the file extension .ppcss, to ensure that it is not treatd like a regular css file, but so that the user is aware of its purpose.

2.Anywhere in the code you want the color to be changed, either type "+main+" or "+secondary+".
